### Chains Manipulation Functions

2)  cutchainspois = Cut a dataframe of chains to obtain chains lengths of a poisson distribution
	- chains = DF with generated chains 
	- meanlen = mean length of chains
	- nchains = number of chains

3) states2varsamps = Transform "frequency of states" values to numerical values (variable vectors)
	- statefreq = matrix (rows indexing chains, columns indexing states)
	- dag = DF with joint parametrization
	- nchains = number of chains

4) states2varsampsjoint = Transform state frequencies to joint probability format
	- statejoint = generated by runchainMS (?)
	- nchains = number of chains


### Generate chains with mutation sampler

5) genchainsMS = Generates multiple chains 
	- len = chain length
	- nchains = number of chains
	- bias = prob of starting at state 1
	- joint = joint DF, needs vars states and joint probability

6) genchainsMSclean = Just output the chains
	- len = chain length
	- nchains = number of chains
	- bias = prob of starting at state 1
	- ms = causal strengths
	- bs = base rates

7) genchainsMSpoislen = Draws chain length from Poisson deistribution (according o Davis & Rehder, 2020)
	- meanlen = mean chain length
	- nchains = number of chains
	- bias = prob of starting at state 1
	- joint = joint DF, joint probability


### Generate predicted response distributions based on chains

8) genjoint = Compute mean joint distribution and joint distributions
	- res = DF/matrix, output from genchainsMS or genchainMSpoislen

9) genrespdistr = Generate DF with response distriutions for each inference
	- res = output from genjoint (previous function)
	- betavar =  prior beta

10) genjointclean = Clean version of genjoint, without unnecessary stuff 
	- samples = DF/matrix output from genchainsMSclean

11) BMSrespdistr = Clean version of genrespdistr, without unnecessary stuff
	- res = output from genjointclean (previous function)
	- betavar = prior beta

12) BMSrespdistr.varexp = Same as previous function but tailored to variability exp
	- res = output from genjointclean (previous function)
	- betavar = prior beta

13) chainstoresps.BMS = Go straight from chains to predicted response distributions, related to BMSrespdistr
	- chains = DF with generated chains 
	- meanlen = mean chain length
	- betavar = prior beta

14) chainstoresps.BMS.varexp = From chains to predicted response distributions but related with BMSrespdistr.varexp
	- chains = DF with generated chains 
	- meanlen = mean chain length
	- betavar = prior beta
