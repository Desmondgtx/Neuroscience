Learning rate (β or α) is a parameter that determines how quickly performane improves with practice. Essentially quantifies the efficiency of the learning process.
Learning rate represents how much benefits each trial provides to the participant in terms of reducing response time (RT) or incrementing accuracy

**In the next equation:** 
$$RT = N^{-β}$$
**Higher (β = 0.6):** Faster learning -  RT decreases MORE faster with each trial
**Lower (β = 0.3):** Slower learning - RT decrease slower

β represents the slope of the learning curve on a logaritmic plot
